# react-redux-boilerplate
just working boilerplate

base requirements:

 * Docker
 * Docker-compose

optionaly:
* fabric / fabric3

instalation:

 * clone project
 * cat .env-example > .env 

### build and run dev:

 * fab build
 * fab start
 * fab migrate
 * fab collectstatic
 * fab runserver
 * fab run_dev
 * ...
